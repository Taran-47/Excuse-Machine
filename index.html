Let's read a useless fun fact



How does it works?

This is an intresting website to find fun facts in two different language.

You can specify the language you want to find the fun fact in and specify the category of fact. Category has two options of Random and Today, Random as justified by name gives a random fact and Today gives a new fun fact every new day.

Default value of Language is set to English and default value of category is set to Random .
